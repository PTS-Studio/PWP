    Проекты

    Внимание! Многие из моих работ будут доступны только на GitHub в связи с работами на сайте.

    Я занимаюсь разработкой сайтов и веб-приложений, обеспечивая их самыми важными функциями и удобствами для использования. В своей работе я опираюсь на адаптивный дизайн и оптимизацию сайта, чтобы каждый смог без труда просматривать интернет-ресурсы, не боясь просадки производительности на сайте.

    В процессе работы мы проектируем макет, верстаем сайт в HTML-форму, адаптируем сайт под мобильные устройства, тестируем сайт на различных браузерах для правильного отображения контента, проводим SEO-оптимизацию и выкладываем сайт в интернет.

    Кроме того, я занимаюсь программированием на таких языках, как Python и C++, работаю с базами данных. Конечно, пока я только развиваюсь в этом направлении и стараюсь разрабатывать программы для автоматизации рутинных задач. Одним из таких задач стало расширение для очистки неиспользуемого пула классов в Vs Code для CSS стилей, которая просматривает неиспользуемые классы и объекты из HTML файла и выделяет ненужные из них в CSS. (Расширение находится на стадии разработки)
